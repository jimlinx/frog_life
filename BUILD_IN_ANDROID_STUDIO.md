# ✅ Build and Install Using Android Studio

## Why Android Studio?

The Gradle wrapper jar file is missing and can only be generated by Android Studio when you sync the project.

## Complete Step-by-Step Instructions

### Step 1: Open Project in Android Studio

1. **Launch Android Studio**
2. **File** → **Open**
3. Navigate to: `/Users/jimlin/Repos/frog_life`
4. Click **Open**

### Step 2: Sync Gradle (CRITICAL!)

When the project opens, you'll see a notification bar at the top:

**"Gradle files have changed since last sync"**

1. Click **"Sync Now"**
2. **Wait for sync to complete** (watch bottom status bar)
   - First sync downloads Gradle wrapper
   - Also downloads all dependencies
   - May take 2-5 minutes

**Don't skip this step!** This generates the gradle-wrapper.jar file.

### Step 3: Clean Project

1. **Build** menu → **Clean Project**
2. Wait for completion (shows in bottom status bar)

### Step 4: Rebuild Project

1. **Build** menu → **Rebuild Project**
2. Wait for build to finish
3. Check "Build" tab at bottom - should say **"BUILD SUCCESSFUL"**

If build fails, check the error messages in the Build tab.

### Step 5: Uninstall Old Version from Device

**On your Android device:**
1. Go to **Settings**
2. **Apps** → Find **"Frog Life"**
3. Tap **Uninstall**

**Or via terminal (if ADB available):**
```bash
adb uninstall com.froglife
```

### Step 6: Connect Device

1. Connect your Android device via USB
2. Enable **USB Debugging** on device (Developer Options)
3. Accept the "Allow USB debugging?" popup on device
4. In Android Studio, check top toolbar - you should see your device name

### Step 7: Run the App

1. Click the green **Run** button (▶️) at the top
2. Or: **Run** menu → **Run 'app'**
3. Select your device from the list
4. Click **OK**

Android Studio will:
- Build the APK
- Install on your device
- Launch the app automatically

### Step 8: Check Device

**After installation:**

1. **Did the app open on screen?**
   - ✅ YES → Great! App is working.
   - ❌ NO → Check "Run" tab at bottom for errors

2. **Go to home screen and open app drawer**
   - Look for **"Frog Life"**
   - Icon should show Android robot on green background

3. **Do you see the icon?**
   - ✅ YES → **SUCCESS!** Everything works!
   - ❌ NO → See troubleshooting below

## Troubleshooting

### If Sync Fails:

1. **Check Internet Connection** - Gradle downloads dependencies
2. **File** → **Invalidate Caches** → **Invalidate and Restart**
3. Try sync again after restart

### If Build Fails:

**Read the error in Build tab carefully.** Common issues:

- **"SDK not found"** → Tools → SDK Manager → Install Android SDK 34
- **"Unsupported Gradle version"** → Update Gradle wrapper in Android Studio
- **Kotlin errors** → Let sync complete fully

### If Install Fails:

- Check device is connected (green device icon in toolbar)
- Ensure USB Debugging is enabled
- Try different USB cable/port
- Restart ADB: `adb kill-server && adb start-server`

### If App Crashes on Launch:

1. Check "Run" tab for stack trace
2. Check "Logcat" tab - filter by "com.froglife"
3. Look for error messages in red

### If Icon Doesn't Appear in Launcher:

**First, verify app is installed:**
- Settings → Apps → Look for "Frog Life"
- Should be in the list

**If installed but no icon:**

1. **Restart your device** (most reliable)
2. **Clear launcher cache**:
   - Settings → Apps → [Your Launcher]
   - Storage → Clear Cache
3. **Try different launcher** temporarily
4. **Check manually**:
   - Settings → Apps → Frog Life
   - Click "Open" - does it work?
